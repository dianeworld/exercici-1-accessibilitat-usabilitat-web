
# Exercici 1 - Accessibilitat i Usabilitat Web

**Autora:** Diana Urbano Fernandez
**Rol:** Dissenyadora Gràfica | UX/UI | Frontend

## Descripció del projecte

Aquest projecte és l'Exercici 1 del curs IFCD0110 Web i està centrat en l’**accessibilitat** i la **usabilitat** en pàgines web. L’objectiu principal és crear un lloc web que sigui **clar, intuïtiu i accessible per al màxim nombre de persones**, independentment de les seves capacitats o del dispositiu que utilitzin per accedir-hi.

El projecte inclou:

- Una pàgina principal (`index.html`) amb enllaços a seccions específiques.
- Seccions detallades sobre **accessibilitat** i **usabilitat**, amb exemples pràctics i referències a experts.
- Imatges descriptives i enllaços a fonts originals.
- Referències a iniciatives i estàndards com el **W3C WAI**.
- Informació sobre experts reconeguts en UX i usabilitat, com **Jakob Nielsen** i **Steve Krug**.
  
## Tecnologies utilitzades

- HTML5 i semàntica adequada (`<header>`, `<nav>`, `<main>`, `<section>`, `<footer>`).
- CSS per a l’estil visual i maquetació responsive.
- Fonts de Google Fonts (`Montserrat` i `Aleo`).
- Pràctiques d’**accessibilitat web** (text alternatiu per a imatges, arxiu estructurat i etiquetes semàntiques).
- Enllaços a referències i recursos externs.

## Com executar el projecte

1. Obre la carpeta del projecte al teu navegador.
2. Obre `index.html`.
3. Navega per les seccions d’**accessibilitat** i **usabilitat** per veure exemples i contingut.

## Observacions

Aquest exercici em permet practicar la creació de pàgines web accessibles i usables, aplicant les recomanacions WCAG i les bones pràctiques de UX/UI. La web és clara, intuïtiva i facilita la navegació amb tot tipus de dispositius i usuaris.
